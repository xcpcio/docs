## Develop

```bash
git clone https://github.com/xcpcio/docs.git -b main --depth=1

cd docs

# install mkdocs-material and plugins
pip3 install -U -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple/

# install node deps
pnpm i

# build
pnpm build

# dev
pnpm dev
```
